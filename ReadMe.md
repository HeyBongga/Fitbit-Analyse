# Fitbit Analyse

Dieses Projekt analysiert und verarbeitet Fitbit-Daten zur Auswertung von Gesundheits- und Aktivitätsmetriken.

## Features

- Datenimport von Fitbit-Geräten
- Analyse von Aktivitätsdaten
- Visualisierung von Gesundheitsmetriken
- Statistische Auswertungen

## Anforderungen

- Fitbit SDK
